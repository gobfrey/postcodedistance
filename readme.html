Overview

This service acts as an interface to the Ordnance Survey Open Postcode Data, available on the OS site. It currently provides easting and northing coordinates and vaguely accurate lat/long coords. It will also provide the distance in miles and metres between two postcodes, and the distance between a postcode and each of a list of postcodes.
